# car-prices-analysis-Python
500,000+ used car records with 16 features including: - Year, Make, Model   - Selling Price ($), Odometer (miles) - Condition (1-50 score), Color, Interior - State, Transmission type  **Analysis includes**: Data cleaning, price statistics, top models/states,  condition analysis, and 7 visualizations (correlation heatmap, trends, boxplots).
